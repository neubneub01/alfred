# Standard Workpaper Format

## Header (Every Workpaper)

```
Client:       [CLIENT NAME]
Engagement:   [Tax Compliance / Provision / Advisory]
Tax Year:     [YYYY] or [FYE MM/DD/YYYY]
Workpaper:    [WP Reference #]
Prepared by:  [INITIALS]    Date: [MM/DD/YYYY]
Reviewed by:  [INITIALS]    Date: [MM/DD/YYYY]
```

## Workpaper Index

| WP # | Description |
|------|-------------|
| 100 | Tax return summary / filing checklist |
| 200 | Income reconciliation (book to tax) |
| 300 | Permanent differences |
| 400 | Temporary differences |
| 500 | Depreciation / fixed assets |
| 600 | State apportionment |
| 700 | Tax provision (ASC 740) |
| 800 | Uncertain tax positions (FIN 48) |
| 900 | Credits and carryforwards |
| 1000 | Miscellaneous / supporting schedules |

## Standard Workpaper Body

### Section 1: Purpose
Brief description of what this workpaper documents and why.

### Section 2: Source Documents
List of PBC items and source data used.

### Section 3: Analysis
The substantive work — computations, reconciliations, analysis.

### Section 4: Conclusion
Summary finding, amount to flow to return/provision, any open items.

### Section 5: Tickmarks

| Symbol | Meaning |
|--------|---------|
| ✓ | Agreed to source document |
| ⊕ | Footed / cross-footed |
| ◊ | Agreed to prior year return |
| △ | Agreed to trial balance |
| PY | Agreed to prior year workpaper |
| N/A | Not applicable — [explain] |

## Review Notes Format

```
[DATE] [REVIEWER INITIALS]: [Note/question]
[DATE] [PREPARER INITIALS]: [Response/resolution]
```

## Quality Standards

- All amounts tied to source with tickmarks
- All formulas visible (no hardcoded numbers without source reference)
- Conclusions clearly stated
- Review notes resolved before finalization
- Workpapers organized in index order
- Cross-references between related workpapers noted
